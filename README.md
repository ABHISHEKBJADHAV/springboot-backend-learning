# Spring Boot Backend Learning Repository

## Overview
This repository contains my structured learning and practice material for backend development using Java and Spring Boot.
It includes hands-on implementations of core Spring concepts, microservices architecture, security, and real-world backend patterns.

## Topics Covered
- Core Java & Advanced Java
- Spring Boot Fundamentals
- Spring MVC & REST API Development
- Spring Security (JWT, OAuth2)
- Microservices Architecture
- Service Discovery (Eureka)
- API Gateway
- OpenFeign Client
- Database Integration with JPA/Hibernate & PostgreSQL
- Exception Handling & Validation
- Docker (Basics)
- AWS Deployment Concepts

## Purpose
This repository reflects my continuous learning and practical backend development journey, showcasing real-world use cases,
design patterns, and system architecture.

## How to Use
Each folder represents a specific topic or mini-project.
You can explore them independently to understand individual backend concepts.
